# ⭐ LifeHub

A personal productivity hub developed in Python.

## About

LifeHub is a personal organization system created to practice Python programming while applying software development concepts such as modularization, code reuse and clean project organization.

The project is being developed incrementally, with each module implementing new programming concepts learned throughout my studies.

## Features

### ✅ Available

* Financial control

  * Register income
  * Register expenses
  * Automatic balance calculation
  * Financial history
  * Summary of income, expenses and current balance

### 🚧 Planned

* Goals management
* Wishlist organization
* Personal routines
* Recipe manager

## Technologies

* Python 3
* PyCharm
* Git
* GitHub
